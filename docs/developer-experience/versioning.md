# Versioning

We adhere to Semantic Versioning (SemVer) `MAJOR.MINOR.PATCH`.

- **MAJOR**: Incompatible API changes, significant architectural shifts, or changes to the vector index schema that require a complete rebuild.
- **MINOR**: Adding new functionality in a backward-compatible manner (e.g., adding a new LLM provider, adding a new document parser).
- **PATCH**: Backward-compatible bug fixes or security patches.
