# Synthetic Enterprise Datasets

All data is synthetic and generated by `data/generate_data.py`. Each artefact is
registered in `data/documents/manifest.json` with its `department`, `sensitivity` and
`allowed_roles`, which the RBAC engine and every chunk inherit.

## Source types

- **PDF** — 15 documents rendered with reportlab; real, queryable prose with page
  markers preserved for citations.
- **CSV** — `operations_metrics.csv`: per-day service metrics (requests, error rate,
  p99 latency, CPU).
- **SQL** — `operations.db` (SQLite): `servers` (hostname, region, role, status, …)
  and `tickets` (id, priority, service, status, summary).
- **JSON** — access logs, security events and audit trail, one event per record.

## Regenerating

```bash
python -m data.generate_data
```

Deterministic and idempotent: safe to re-run. The vector index is rebuilt from the
manifest on the next `build_index()` / API startup.
