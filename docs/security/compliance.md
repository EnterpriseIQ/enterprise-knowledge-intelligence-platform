# Compliance

EnterpriseIQ is designed to help organizations meet compliance requirements like SOC2, GDPR, and HIPAA regarding data access controls.

## Explainability
Compliance requires knowing *why* a decision was made. Every query response includes an `access_summary` detailing exactly how many documents were retrieved, how many were allowed, and how many were denied by RBAC.

## Grounded Generative AI
By explicitly citing source documents for every generated sentence and providing an honest confidence score, EnterpriseIQ complies with enterprise mandates against "black-box AI."
